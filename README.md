# Job Portal Backend

Backend scaffold for a job portal with role-based flows for admin, employer, and employee users.

## Structure

```text
src/
  config/
  models/
  schemas/
  controllers/
  routes/
  services/
  middlewares/
  utils/
  constants/
  validations/
  app.js
  server.js
uploads/
logs/
tests/
```

## Run

```bash
npm install
npm run dev
```
